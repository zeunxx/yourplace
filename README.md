#### ✍️ 주요기능 : 
1. 회원 가입 및 로그인
2. 비밀번호 찾기 및 비밀번호 변경 기능
3. 마이페이지에 찜 목록 출력
4. 메인 페이지
    - 오늘의 날씨
    - 날씨 맞춤 식당, 카페 추천
5. 식당, 카페 검색
6. 검색 후 출력된 식당/카페 리스트 클릭시 세부사항 페이지 출력
    - 운영 시간, 전화번호 등
